4. Agent Workflow

  1. Read the local repository instructions.
  2. Inspect the relevant files before editing.
  3. Make the smallest useful change.
  4. Run the smallest meaningful verification.
  5. Report changed files, verification, and remaining risk.

5. MCP Policy

The default project MCP file is empty.

{
  "mcpServers": {}
}

Use built-in Claude Code and Codex tools first. Add project-scoped MCP only when the task has a concrete need.

ToolPolicy
openaiDeveloperDocsAllowed globally in Codex for OpenAI official docs.
@playwright/mcp@latestUse project-scoped for browser verification.
context7Add only temporarily when library docs are a real blocker.
brave-searchNot needed by default because built-in web search is available.

6. Verification

Documentation-only changes:

git diff --check

Node product changes:

pnpm lint
pnpm build
pnpm test

Browser-facing changes may also require Playwright tests or a browser smoke check.
